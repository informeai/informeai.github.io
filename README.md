## 🚀 Como Executar

1. **Clone o repositório**
   ```bash
   git clone https://github.com/informeai/informeai.github.io.git
   cd informeai.github.io
   ```

2. **Abra o arquivo**
   - Abra `index.html` em qualquer navegador moderno
   - Ou use um servidor local:
     ```bash
     python -m http.server 8000
     # ou
     npx serve .
     ```

3. **Acesse**
   - Abra `http://localhost:8000` no navegador

## 🔧 Personalização

### Alterar Cores
Edite as variáveis CSS em `css/style.css`:
```css
:root {
  --primary-color: #0071e3;
  --primary-hover: #0077ed;
  --text-primary: #1d1d1f;
  /* ... outras variáveis */
}
```

### Alterar Conteúdo
- **Informações pessoais**: Edite o HTML em `index.html`
- **Serviços**: Adicione/remova cards na seção de serviços
- **Habilidades**: Modifique as categorias e tecnologias
- **Contato**: Atualize o email de contato

### Alterar Estilos
- **Layout**: Modifique o CSS em `css/style.css`
- **Animações**: Ajuste as transições e keyframes
- **Responsividade**: Edite as media queries
